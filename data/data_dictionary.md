# Data Dictionary

| Field | Description |
| --- | --- |
| customer_id | Synthetic customer identifier |
| customer_segment | Retail banking customer segment |
| region | UK region |
| tenure_months | Customer tenure with the bank |
| product_count | Number of active products |
| digital_logins_30d | Digital banking logins in last 30 days |
| app_sessions_change_pct | Month-on-month digital session change |
| salary_inflow | Whether salary is paid into the account |
| salary_inflow_change_pct | Change in salary inflow value |
| avg_balance | Average account balance in GBP |
| balance_change_pct | Month-on-month balance change |
| direct_debits | Number of active direct debits |
| complaint_count_90d | Complaints in last 90 days |
| overdraft_usage_rate | Share of overdraft limit used |
| card_spend_30d | Card spend in last 30 days |
| open_banking_connected | Open banking connection indicator |
| churn_flag | Simulated churn outcome |
| churn_risk_score | Explainable churn score generated by this project |
| churn_risk_band | Low, medium, or high churn risk |
| retention_action | Recommended retention treatment |
| value_at_risk | Balance value associated with simulated churn |
